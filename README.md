# console-tetris

使用 node.js 开发，在 Command Line 进行交互的俄罗斯方块游戏。因为依赖 Unicode emoji，程序目前仅支持 Mac Terminal，不完全兼容 Powershell。
## Getting Started

### 用法
```js
npx console-tetris
```
或
```js
git clone https://github.com/rhythm022/console-tetris
cd console-tetris
npm i
npm run play 
```


### 游戏控制
```
左移：方向键 左
右移：方向键 右
变形：空格
加速：方向键 下
落地：方向键 上
```
