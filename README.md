# console-tetris

使用 node.js 开发，在 Command Line 进行交互的俄罗斯方块游戏。
## Getting Started

### Usage
```js
git clone https://github.com/rhythm022/console-tetris
cd console-tetris
npm i
npm run play 
```


### 游戏控制
```
左移：方向键 左
右移：方向键 右
变形：空格
加速：方向键 下
落地：方向键 上
```
